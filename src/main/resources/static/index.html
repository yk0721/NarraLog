あなたの物語を もっと深く、共感。

NarraLogは、日々のボヤきや悩みを共有し、他者との対話を通じて新しい視点を見つけるためのコミュニケーションプラットフォームです。

Features

新しい対話の形

ボヤき投稿

誰にも言えないちょっとした悩みや、心に留まった出来事を気軽に投稿できます。

リアルタイムチャット

共感したユーザーと即座にチャット。リアルタイムなやり取りで深まる理解。

共感の輪

同じ悩みを持つ人々とつながり、孤独を感じない居場所を作ります。

NarraLogのミッション

私たちは、一人ひとりの「ナラティブ（語り）」を大切にしています。 言葉にすることで整理され、誰かに届くことで力になる。そんな体験を届けるために、NarraLogは生まれました。
