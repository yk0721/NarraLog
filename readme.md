# NarraLog ~ナラログ~

### プロジェクトについて
このプロジェクトは同じことで困っている人々をマッチングして，その困りごとを解決する過程をナラティブ(物語)化してログ(記録)していくツールです。

Yahoo知恵袋やQuoraといったサービスは「一対多」で問題を解決し，似たようなことで困っている人々はその問題解決した記録を俯瞰するだけになります。
それに対してNarraLogでは「多対多」で問題を解決し，その解決の過程をナラティブ(物語)化してログ，ナレッジベースを作成します。

このプロジェクトだと，問題の提起者も含め，困っている人，それを解決する能力を持っている人が一緒になって問題に対処するため，その問題についてより深い考えを共有し，提起当初よりも深い理解度で問題に対処していくことが可能です。

また，多数の有識者が会話に参加することによって，解決者としても単一の視点からではなく，複数の視点からの解決策を提供することができます。

### 使用方法
このシステムは以下の構成で構築されています。

- PostgreSQL (Docker上にて動作)
- SpringBoot (Gradleによりビルド)
- React + Vite

### デプロイ方法

1.Docker-Composeを利用してPostgreSQLサーバーを起動します。
`docker-compose up`

2.アプリケーションをビルドします。
`./gradlew build`

3.アプリケーションを起動します。
`./gradlew bootRun`

4.frontendフォルダに移動して，
`cd frontend`

5.Viteでアプリケーションを起動します。
`npm run dev`
